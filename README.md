# GitHub-Jenkins-SonarQube-Docker-DockerHub-Kubernetes-EKS-Terraform-Infra-
Production-ready DevOps project implementing automated CI/CD pipeline with Jenkins, Docker, Kubernetes (EKS), and Terraform for scalable cloud deployment.
# 🚀 DevOps CI/CD Pipeline Project

## 📌 Overview
This project demonstrates a complete DevOps pipeline that automates application deployment using Jenkins, Docker, Kubernetes, and Terraform on AWS.

## ⚙️ Features
- CI/CD automation using Jenkins  
- Docker containerization  
- Kubernetes deployment  
- Infrastructure as Code using Terraform  

## 🏗️ Architecture
GitHub → Jenkins → Docker → DockerHub → Kubernetes → AWS

## 🛠️ Tools Used
- Jenkins  
- Docker  
- Kubernetes  
- Terraform  
- AWS  

## 🚀 Setup Steps
1. Clone repo  
2. Build Docker image  
3. Push to DockerHub  
4. Deploy using Kubernetes  

## 📌 Outcome
- Automated deployments  
- Reduced manual effort  
- Scalable architecture  

## 📌 Conclusion
This project showcases real-world DevOps practices including CI/CD, containerization, and cloud automation.
